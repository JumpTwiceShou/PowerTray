This release focuses on bug fixes, stability improvements, and packaging cleanup.

## Highlights

- Improved native Logitech device detection and offline status handling.
- Added GitHub release update checking.
- Added bilingual settings, diagnostics, theme support, and per-device alert options.
- Reduced installer size with separate lightweight and full installer options.
- Fixed tray menu, settings layout, duplicate shortcut, tooltip, and dependency vulnerability issues.

## Downloads

- `PowerTraySetup.exe`: lightweight installer, requires .NET 8 Desktop Runtime x64.
- `PowerTrayFull.exe`: full installer with runtime included.
