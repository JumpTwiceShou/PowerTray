This release improves native Logitech device detection speed, USB disconnect/reconnect stability, and update-check preferences.

## Highlights

- Improved native device detection speed after startup, manual rediscovery, and USB receiver reconnects.
- Improved USB disconnect/reconnect handling so known devices go offline and return online more reliably.
- Improved PRO X2 SUPERSTRIKE Wireless Mouse detection stability in native-only mode.
- Reduced C54D LIGHTSPEED receiver recovery delays and unnecessary HID++ retries.
- Added automatic update checks, configurable in Settings, with download confirmation before installer download.

## Validated devices

- PRO X2 SUPERSTRIKE Wireless Mouse
- PRO X 2 Lightspeed Gaming Headset

## Downloads

- `PowerTraySetup.exe`: lightweight installer, requires .NET 8 runtime components.
- `PowerTraySetup-full.exe`: full installer with runtime included.
