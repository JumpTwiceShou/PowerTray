# PowerTray

PowerTray is a native-only Logitech battery tray app for Windows, based on a vibe-driven modification and optimization of [andyvorld/LGSTrayBattery](https://github.com/andyvorld/LGSTrayBattery).

PowerTray 是一个 native-only 的 Windows 罗技设备电量托盘工具，基于 [andyvorld/LGSTrayBattery](https://github.com/andyvorld/LGSTrayBattery) 进行 vibe 修改和优化。
